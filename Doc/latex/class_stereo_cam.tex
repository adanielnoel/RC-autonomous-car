\subsection{Detailed Description}
Class that manages stereo cameras. The \hyperlink{class_stereo_cam}{Stereo\+Cam} class suports both generic and D\+U\+O3D stereo cameras. It provides access to images from the device, as well as methods for generating calibration and rectification parameters and apply them to the retrieved images.
